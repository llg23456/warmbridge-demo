## 4. 通俗视频生成（MVP-C）流水线

**不做**：下载原片、ASR、图生视频。

```
条目(孩子推荐/快解析, 抖音/B站)
  → prepare: 拉 og 封面 → cover_raw.bin → frame JPEG
  → script: explain(缓存) → LLM 口播稿 45～75s（含背景/词语）
  → tts: vivo §14 → narration.wav
  → merge: silent.mp4(静态图循环) + aac → {job_id}.mp4
  → done: /api/video/popular/files/{job_id}.mp4
```

| step | 进度 | 服务端文件 |
| --- | --- | --- |
| prepare | 5% | `server/data/popular_videos/{job_id}/cover_*` |
| script | 30% | — |
| tts | 55% | `narration.wav` |
| merge | 80% | `silent.mp4` → `{job_id}.mp4` |
| done | 100% | 对外 URL |

**日志 tag**：`WbVideoGen`（服务端 + Android Logcat）

## 8. 已知问题与现象（2026-06-01）

| # | 现象 | 原因 / 处理状态 |
| --- | --- | --- |
| 1 | 视频画面是**橙色底 + 白色方框** | 封面下载失败走占位图；`drawtext` 写中文标题但 **ffmpeg 无中文字体** → 方框。修复：占位图去掉中文 drawtext；增强封面下载 Referer |
| 2 | 只有封面占位、不是视频原封面 | B 站/抖音 og 图防盗链，需带 Referer / 更长超时；失败时用无字品牌占位图 |
| 3 | 任务 / 分享数据重启丢失 | 设计为 Demo 内存存储 |
| 4 | `video_url` 换 Wi‑Fi 失效 | 含局域网 IP，换网需重生成 |
| 5 | 口播过长可能截断 | TTS 单段约 2048 字节上限，超长口播会被 `vivo_tts` 裁剪 |
| 6 | 协程取消误报网络 | 已过滤 `left the composition`（`humanizeNetworkError`） |
| 7 | 多次生成后变卡 | 服务端：任务结束删 work 目录；仅保留最近 **10** 个 mp4（超 6h 删）。App：离开通俗视频页 / 保存相册后清 `cache/wb_popular_*.mp4` |

## 10. 下一步建议（视频画面）

1. 占位图：**纯橙色 + 简单图形**，不写中文（避免方框）。
2. 封面下载：请求头加 **Referer: 视频页 URL**、Accept 图片。
3. 可选：服务端内置 `server/assets/default_cover.jpg` 作为最终兜底。
4. 日志：prepare 阶段打印 `cover_url`、下载字节数、是否 placeholder。
